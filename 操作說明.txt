IPQC Smart Factory V2.1 記住登入版

新增功能：
- 第一次登入成功後，登入狀態會保存在目前裝置。
- 關閉網頁、重新開啟或從手機桌面進入時，會自動登入。
- 只有按「登出」、清除瀏覽器網站資料，或登入狀態失效時，才需重新輸入密碼。

更新方式：
1. 解壓縮 ZIP。
2. 將全部檔案上傳到 GitHub Repository 根目錄。
3. 覆蓋同名檔案並 Commit changes。
4. 等候 GitHub Pages 約 1～3 分鐘。
5. 第一次更新後，建議重新整理兩次；手機若仍顯示舊版，可關閉 App 後再開啟。

注意：
- 每一台新手機或新電腦第一次仍需登入一次。
- 無痕模式通常不會永久保存登入狀態。
- 主動按登出後，下次需要重新登入。

IPQC Smart Factory V2.0 可直接試用版

本版已直接連接你的 Supabase，不需要修改任何檔案。

上傳方式：
1. 解壓縮 ZIP。
2. GitHub Repository → Add file → Upload files。
3. 把解壓縮後全部檔案拖入並覆蓋舊檔。
4. 按 Commit changes。
5. 等待約 1～3 分鐘。
6. 開啟：
   https://chengyin1024.github.io/IPQC-Smart-Factory/

登入：
使用你在 Supabase Authentication 建立的 Email 與密碼。

測試同步：
1. 電腦新增人員、料號與巡檢。
2. 手機開同一網址登入。
3. 查看是否出現相同資料。
4. 手機新增巡檢，再回電腦查看。

目前可用：
- 登入
- Dashboard
- 人員管理
- 料號管理
- 新增巡檢
- 巡檢紀錄
- 99% 酒精
- 3M600
- 雲端共用資料
